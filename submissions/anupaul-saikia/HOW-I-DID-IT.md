## What I did 

> In blender, I created a 3D mockup of a wind tunnel, live date telemetrics using geometry nodes

> I also made a walkthrough video, provided as a link.

> Also made an agent that that takes blender format and converts it to FBX format, which is more supported by other CAD/M Softwars. Addition to this for making it as a true "agent" 

## Problems I found

> Blender isn't the ideal software for working on real-time simulations, with tinkering it does support plugins to produce real-time values
