## Agent Name
Health Advisor Agent

---

## Problem Statement
Many people experience fatigue, poor sleep, and unhealthy lifestyle habits but do not understand the cause or how to improve.

This agent helps users identify possible reasons and provides actionable suggestions.

---

## What the Agent Does
- Takes a user query
- Identifies key issues (sleep, nutrition, lifestyle)
- Uses knowledge tools
- Provides suggestions

---

## Tools Used (LPI Conceptual Tools)
- Sleep Knowledge Tool → Provides sleep guidelines
- Nutrition Tool → Provides diet insights

---

## How It Works
1. User enters a query
2. Agent identifies problem area
3. Agent queries tools
4. Combines results
5. Returns advice

---

## Example Input
"I feel tired and don’t sleep well"

## Example Output
- Improve sleep duration (7–9 hours)
- Increase protein intake
- Stay hydrated
- Maintain a routine

---

## Explainability
The agent explains recommendations by linking them to sleep and nutrition issues.

---

## Limitations
- Uses static knowledge
- Not personalized yet
